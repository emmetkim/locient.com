왜 필요한가

스마트폰으로 40명을 관리할 수 없는
4가지 이유

📱

스마트폰이 없다

초등 저학년은 미보유. 보유 학생도 데이터 부족, 부모의 앱 설치 제약.

⏱️

40명 앱 설치 = 1시간

앱 다운로드→계정→위치 권한→동의. 40명이면 활동 시간보다 설정 시간이 길다.

🔋

배터리가 죽는다

GPS 앱은 배터리를 빨아먹는다. 중고생은 GPS를 끈다. 4시간 후 절반이 오프라인.

👴

어르신은 앱을 못 깔아

단체 관광·복지시설. 앱 설치를 요구하는 순간 반이 포기한다.

2025년 학교안전법 개정 이후, 서울 초등학교 현장체험학습 실시율이 98.8%에서 51.1%로 급감.
교사 면책에 "안전조치 이행 증명"이 필수화되었으나 — 물리적으로 기록할 수단이 없다.

사용 사례

이런 현장에서 씁니다

⛰️

산악 가이드 — 20명 산행

태그 NFC 탭으로 40초에 전원 등록. 선두에서 리더로 뒤처진 참가자 실시간 확인. 숲속은 태그 릴레이로 4km 커버. 하산 후 위치 로그→PDF 안전 기록 자동 생성.

🚴

자전거 대회 — 40명 코스 모니터링

운영 본부 리더로 참가자 전원 코스 진행 확인. 낙차 감지 시 SOS 1초 전달 + 정확한 위치. 대회 후 결과 보고서 자동 생성.

🏕️

수련원 야외 활동 — 30명 오리엔티어링

원장이 태그 배포하며 NFC 탭(30명 1분). 숲속 이탈 시 자동 경보. PDF 안전관리 보고서→교육청 제출·교사 면책 소명 자료.

🧓

치매 관제 — 주간보호 시설 야외 나들이

어르신에게 태그를 클립으로 부착. 앱 설치·조작 불필요. 시설 반경을 벗어나면 즉시 경보. 위치 기록이 자동으로 남아 보호자 보고에 활용.

🥾

개인 등산/자전거 — 리더 단독 모드

태그 없이 리더만 사용. 속도·고도·경로 기록 + 출발지 복귀 내비. 인터넷 없이 산에서도 작동. 별도 기기 없이 GPS 컴퓨터로 전환.
