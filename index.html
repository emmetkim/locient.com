— 군집 모드 —

리더 — 팀원 전원의 위치

지도 위에 태그 착용자(T1~T5)의 GPS 위치가 실시간 갱신됩니다. 경로를 이탈한 T3에 대한 경보가 하단에 즉시 표시됩니다.

태그 — 리더가 어디에 있는지

리더의 실제 이동 경로가 선으로 표시되고, 나와 리더 사이의 거리(320m)가 큰 숫자로 보입니다.

— 단독 모드 —

라이딩 대시보드

현재 속도, 이동 거리, 시간, 고도, 누적 상승/하강을 한 화면에. 하단에 이동 경로와 고도 프로필이 표시됩니다.

출발지 복귀

나침반으로 출발지 방향을 가리키고, 직선 거리를 큰 숫자로 표시. 하단 지도에 내가 온 경로와 출발지가 보입니다.

작동 방식

복잡한 설정 없이, 3단계면 끝

1

태그를 나눠주세요

참가자에게 태그를 건네고 전원만 켜면 됩니다. 리더 근처에서 자동 등록.

2

리더에서 확인하세요

지도 위에 팀원 전원의 위치가 점으로 표시됩니다. 인터넷 없이 작동합니다.

3

경보가 알려줍니다

경로 이탈, 넘어짐, SOS — 리더가 진동과 소리로 즉시 알려줍니다.

주요 기능

관리자가 알아야 할 것만

📍

실시간 위치 추적

10초마다 전원의 GPS 위치가 지도 위에 갱신됩니다.

⚠️

이탈 자동 경보

경로나 구역을 벗어나면 리더에 자동 알림.

🆘

SOS 긴급 호출

측면 버튼 3초 장누르기로 1초 내 리더에 위치 전달.

🤸

낙차 자동 감지

넘어짐을 센서가 감지, SOS 없이도 자동 경보.

📡

인터넷 불필요

자체 무선통신. 산속, 계곡, 섬에서도 작동. 통신비 없음.

👥

대규모 연동

리더 여러 대 연결 → 총괄 리더 1대에서 최대 400명 통합 관제.

🚴

속도 · 고도 · 경로 기록

단독 모드에서 자전거 컴퓨터/산악 GPS로 활용. 추가 비용 없음.

🔙

올라온 경로를 자동 기록, 방향과 거리를 안내하여 출발지로 복귀.
